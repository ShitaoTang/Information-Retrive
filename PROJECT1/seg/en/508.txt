 network working group john mcconnell request for comments ames research center category d april ames graphics facilities at ames research center this note is intended as a response to an rfc which suggested that nodes describe their graphics environment the following graphics facilites are now at ames for the ibm tss a an sc off line microfilm system b calcomp plotters c ards storage tube crts d imlac pds refresh crts e one ibm crt devices which will be in use in the near future are a an evans and sutherland lds system b hazeltine alphanumeric crts because of the diverse nature of the devices and the varying software support provided by the manafacturers work has been undertaken to provide a device independent package this package is currently being implemented the basic entity is a device independent intermediate language which describes the picture to be displayed when a device is selected the intermediate language is converted to specific device orders other features include the definition and storage of subpictures for later construction of more complex pictures an interface has been specified so that graphic data sets can be transmitted to other installations on the tss network a note to those who plan to use the illiac iv when it becomes a functioning node at ames i would like information concerning the graphics environment at your sites and any thoughts you have on how you will be using graphics with the illiac iv this rfc was put into machine readable form for entry into the online rfc archives by andrew crawford page
