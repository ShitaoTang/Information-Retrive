 arpa network information center stanford research institute menlo park california nwg rfc jonathan b postel ucla nmc nic dec character generator process i hereby propose that there be a standard process implemented on whatever hosts desire which generates character data with out any regard to input in some sense the process is the converse of the discard process rfc many hosts have an existing terminal testing program e g ttytst which would suit this function adequately this character generator process would listen for a request for connection and execute the initial connection protocol icp as specified in nic the current network protocols notebook upon completion of the icp the character generator process would begin to send characters into the network as buffer space is made available the character generator process is terminated by closing the connections this rfc was put into machine readable form for entry into the online rfc archives by bbn corp under the direction of alex mckenzie page
