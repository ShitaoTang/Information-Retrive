means whereby the user identifies himself and specifies the account number to which his use of smfs is to be billed the user should issue a lgi command directly after completing the icp and before any command referencing a file the user name and account number specified remain in effect until another lgi command is issued a lgo command is issued or the connection is closed at present the use of smfs is not billed and therefore use of the accounting commands is optional it is requested however that users and user processes begin to use this command as soon as possible since we would like to collect statistics on smfs utilization before implementing billing therefore at present the user name can be any name that identfies the user and the account number is completely arbitrary the format
